hexo部署到github

發表於 2015-12-25

####在已经搭建号hexo的前提上，进行如下的操作来实现部署到github上

  1. 首先注册一个github账号，
  2. 创建与github账号同名的仓库,如sunny-lee24.github.io
  3. 拷贝仓库的url，如：https://github.com/sunny-lee24/sunny-lee24.github.io.git
  4. 打开终端，安装hexo-deployer-git
    $ npm install hexo-deployer-git save
  5. 修改配置文件_config.yml
    $ vim _config.yml
        deploy: 
type: git
repo: https://github.com/sunny-lee24/sunny-lee24.github.io.git
branch: master
    注意，在修改属性时，在冒号“:”后面必须要有一个空格。
    其中repo为仓库的url，branch为分支名。
  6. npm install
  7. hexo g（合成）
  8. hexo d (在第一次进行此步操作时会让输入用户名和密码，此时的用户名和密码是github的用户民和密码)

注意：
每进行一次配置文件的更改都需要进行6、7、8步的操作。
